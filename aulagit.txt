*TEORIA:*

GIT -> programa -> VERSIONA (separa o cód. por versões)

GIT HUB -> plataforma -> HOSPEDA

GIT INIT -> inicialização 

GIT ADD -> adiciona os arquivos

GIT COMMIT -> mensagem

GIT REMOTE SET-URL ORIGIN -> PythonLisboa pythonAulas

GIT PUSH -> enviar
